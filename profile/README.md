## Built Around Real Trading

AutoTrading은 데모 수준의 자동매매 저장소가 아닙니다.  
**한국투자증권 API를 활용한 실제 매매 실행**, **다양한 전략 세팅과 반복 검증**,  
그리고 **운영 가능한 화면과 도구**까지 하나의 흐름으로 연결하는 프로젝트 조직입니다.

세부 기술 스택, 아키텍처, 실행 방법, 전략 로직은 각 repository에서 확인할 수 있습니다.

## Why This Organization Feels Different

- **실매매를 전제로 설계합니다.**  
  연구용 코드에서 끝나지 않고 실제 주문 연결과 운영 안정성까지 고려합니다.

- **전략 변경에 유연합니다.**  
  다양한 전략 세팅과 실험 흐름을 통해 아이디어를 빠르게 검증하고 발전시킵니다.

- **제품처럼 다룹니다.**  
  실행 엔진만이 아니라 UI, 운영 경험, 모니터링, 구조화된 설계까지 함께 만듭니다.

- **연구와 운영을 분리하지 않습니다.**  
  아이디어, 백테스트, 구현, 실행, 관측을 하나의 시스템으로 이어갑니다.
